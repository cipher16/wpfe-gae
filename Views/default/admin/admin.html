Si vous venez pour une première installation :

  1. Uploadez votre export wordpress au format XML
  2. Allez dans l'import et choisissez le dernier import uploadé
  3. Profitez de votre frontend
Ou bien vous pouvez utiliser le système de requête XML RPC (à activer sur votre blog)
  1. Autorisez les requêtes XML RPC en écriture sur votre blog (partie administration)
  2. Créez un utilisateur sur votre blog spécifique a WPFE (histoire d'avoir un autre compte que celui de l'administrateur)
  3. Changez le login/mot de passe dans la configuration liée au XML RPC
  4. Cliquez sur le lien RPC Update, et mettez à jour ce que vous voulez.

Si vous venez pour administrer votre blog ...

Il faut savoir que ceci n'est pas un blog, mais un FRONT END pour wordpress, destiné à réduire la charge de votre serveur.
Les actions possibles sont :
  • L'importation de nouveaux articles par flux RSS
  • Le nettoyage de votre base de données
  • Le nettoyage du cache (memcache)
  • L'utilisation en CDN pour les médias
  • ...
